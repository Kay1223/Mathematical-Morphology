# Mathematical Morphology

Mathematical Morphology (MM) is a theory and technique for the analysis and processing of geometrical structures, based on set theory, lattice theory, topology, and random functions.

** Basic operators **

  1. Erosion
  2. Dilation
  3. Opening
  4. Closing

**References**
